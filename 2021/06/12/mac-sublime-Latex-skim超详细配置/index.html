发表于 2021-06-12 更新于 2021-07-15 Valine：

之前想学一下latex,但是环境没配好，不了了之，今天心血来潮，必拿下！

准备

在 Sublime Text 和 mac 中使用 LaTeX 需要四个工具：

  • MacTeX
  • Sublime Text
  • LaTeXTools
  • Skim

MacTeX 包含在Mac上使用LaTeX所需的与TeX相关的程序和包。Sublime Text是一个用于编写LaTeX文档的文本编辑器。Skim 是用于查看LaTeX文档的PDF查看器。LaTeXTools插件提供了多种功能，可简化使用LaTeX文件并用于构建LaTeX文档。
latextools文档: latextools

step4

安装LaTeXTools

  1. 在sublime中按 ⌘(Command) + Shift + P
  2. 搜索“Package Control: Install Package”
  3. 按 Enter
  4. 搜索“LaTeXTools”
  5. 按Enter

提示以下信息表明安装成功：

step5

安装Skim:skim

step6

配置skim

  1. 打开skim
  2. 从菜单中选择“Skim”，然后选择“Preferences”
  3. 在右上角两个大于号那选择“同步”
  4. 勾选“检查文件更改”
  5. 勾选“自动重新加载”
  6. 从“预设”菜单中选择”sublime text”
